# Sissi Studio - Magazin Online Django

## Descriere Proiect

Sissi Studio este un site de tip magazin online construit cu Django și PostgreSQL. Proiectul demonstrează implementarea unei platforme complete cu funcționalități backend și interfață responsive pentru utilizatori.

## Funcționalități Principale

- Catalog de produse cu detalii și imagini
- Căutare și filtrare produse (după categorie, preț, mărimi)
- Sortare și paginare produse cu AJAX
- Coș de cumpărături
- Autentificare și profil utilizator personalizat
- Management produse (adăugare cu calcul automat de preț)
- Sistem de contact cu validări avansate și salvare JSON
- Newsletter subscription
- Design responsive (desktop și mobile)
- Admin panel pentru gestiune categorii și produse
